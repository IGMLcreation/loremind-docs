# Generating Lore Pages

Beyond free chat, DM Loremind lets you **generate or regenerate a specific
field** of a Lore page through a dedicated button.

## Workflow

1. Open a Lore page (e.g. "City of Vaeril")
2. Next to each TEXT field, a **✨ Generate** button
3. The AI receives as context: the page itself + the other fields already filled in + the parent Lore
4. It proposes text for the targeted field
5. You can **accept**, **regenerate** (a new proposal), or **refine**

## Why field by field and not in bulk?

Targeted generation:

- The AI stays consistent with what is already there
- You keep control field by field
- You can start with an almost empty page and fill it in progressively

## Affected fields

- **Text** only — Image / Key-value list / Table fields are not generated by the AI
- The AI can rely on the value of the other **text** fields as context

## Tips

- **Fill in a few fields by hand first** — this gives the AI something to work with
- **Regenerate several times** — the first proposal is not always the best
- **Refine through chat** — if a generation is almost right, open the chat on the page and ask for the precise adjustment
