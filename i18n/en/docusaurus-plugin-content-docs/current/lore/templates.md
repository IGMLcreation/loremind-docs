# Templates and field types

A template is used to define the structure of a page type.
For example, if you want to define a city with a "History" field and a "Population" field...

## Creating a template

To create a template, click the "+" button in the template header at the bottom of the list
from your lore:

![new template button](/img/screenshots/lore/templates/bouton_new.png)

A new page with a basic template will then open in the middle of the screen:

![new template](/img/screenshots/lore/templates/nouveau_template.png)

### Available field types

| Type | Usage | Rendering |
|------|-------|-------|
| **Text** | Free text, paragraphs (History, Description) | Textarea |
| **Image** | Image gallery (illustrations, maps) | Configurable gallery |
| **Key/value list** | Label → value pairs, labels fixed at the template level (e.g. Population, Area, Climate) | List of label/value fields |
| **Table** | Tabular data in rows and columns (e.g. resources, statistics) | Editable table |

For **Image** fields, several **layouts** are available:

- **GALLERY** — grid of thumbnails (default)
- **HERO** — first image as a banner, the following ones small
- **MASONRY** — mosaic with variable heights
- **CAROUSEL** — horizontal scrolling

:::tip
To change a field's type, use the **dropdown menu** next to the field
(Text, Image, Key/value list, Table).
You can also reorder the fields with the **↑** / **↓** buttons next to each one.
:::

## Editing a template

You can simply edit a template by clicking on it in the template section.

## Deleting a template

You can delete a template by clicking the "Delete" button in the top right when you
have returned to its page.

## Best practices

- **Start minimal** — 3-4 fields are enough to get started; you can add more later.
- **Name clearly** — "History" is more meaningful than "Long description".
- **Image = HERO in first position** — gives the page visual appeal.
