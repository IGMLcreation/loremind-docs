# Welcome to the DM Loremind documentation

**DM Loremind** is a self-hosted tabletop RPG assistant that combines campaign management, character sheets, lore
and local AI to help you prepare your games.

## Key features

<IconFeatureList>
  <IconFeature Icon={Dices} title="Multi-system">manage several campaigns each with their own TTRPG rules.</IconFeature>
  <IconFeature Icon={Sparkles} title="Local or remote AI">embedded Ollama or remote AI as you prefer (currently 1min.ai supported).</IconFeature>
  <IconFeature Icon={ShieldCheck} title="100% self-hosted">your data stays on your machine.</IconFeature>
  <IconFeature Icon={Container} title="One-command Docker install">Windows, Linux, macOS.</IconFeature>
  <IconFeature Icon={RefreshCw} title="Automatic updates">optional, via Watchtower.</IconFeature>
</IconFeatureList>

## Architecture

DM Loremind is made up of three services that communicate through Docker Compose:

- **Core** — Java/Spring Boot backend (business API, Postgres + MinIO persistence).
- **Brain** — Python/FastAPI service dedicated to AI (LLM orchestration, RAG).
- **Web** — Angular user interface.

Optionally, a fourth **Ollama** service can be started to serve the LLM models locally.
