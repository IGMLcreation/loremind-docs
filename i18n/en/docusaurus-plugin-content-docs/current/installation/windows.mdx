# Installation on Windows

![DM Loremind Windows installer](/img/screenshots/install_windows/Install_windows.png)

The Windows installer automates the entire setup: WSL2, Docker Desktop, generation of the secure `.env` file, and launching the stack.

## Prerequisites

- Windows 10 (version 2004+) or Windows 11.
- An account with administrator rights.
- An Internet connection to download Docker Desktop and the images.

<Callout type="warning" title="Hardware requirements for local AI">
To run the AI locally via Ollama, plan for a PC with **at least 16 GB of RAM** and a modern processor / graphics card.

With a lightweight model such as `gemma3:e2b`, 16 GB of RAM and a recent CPU/GPU are enough, but inference will remain slow. For real comfort, aim for 32 GB of RAM and a recent NVIDIA card.
</Callout>

## Procedure

1. **Download the files** from the [GitHub repo](https://github.com/IGMLcreation/LoreMind/tree/main/installers) into a single folder (for example `Downloads\DM Loremind\`):
   - [`install.bat`](https://raw.githubusercontent.com/IGMLcreation/LoreMind/main/installers/install.bat) — launcher
   - [`install.ps1`](https://raw.githubusercontent.com/IGMLcreation/LoreMind/main/installers/install.ps1) — main script
   - [`secure-host-ollama.ps1`](https://raw.githubusercontent.com/IGMLcreation/LoreMind/main/installers/secure-host-ollama.ps1) — *only if you already have Ollama installed on your PC*

2. **Right-click `install.bat`** → **Run as administrator**.

3. Accept the UAC prompt.



## What the installer does

1. Checks / installs **WSL2** (a reboot may be required — relaunch the script afterwards).
2. Checks / installs **Docker Desktop** via `winget`.
3. This is the most important part: the program asks you to choose the username / password
for your administrator account. The browser will ask you for these credentials when you log in to the application
(this is done to secure the settings section, among other things).
4. When you are asked to choose between ollama and 1min.ai during installation,
you can leave it empty. Ollama will be used first by default.
5. Once this is done, the application will attempt to install ollama if you wish: see the
[Configure Ollama](./ollama) section (local LLM engine) of the documentation.
6. Generates `%LOCALAPPDATA%\DM Loremind\.env` with random passwords.
7. Launches the stack and opens `http://localhost:8081` in your browser.

<Callout type="info" title="Why install.bat?">
The `.bat` file is only there to launch `install.ps1` cleanly (UAC + an ExecutionPolicy scoped to the session, without modifying system settings). It is purely declarative and can be audited in just a few lines.
</Callout>

## Antivirus false positives

Some antivirus programs may flag PowerShell scripts. If that happens:

- Verify that you are indeed downloading from the official organization [`IGMLcreation/LoreMind`](https://github.com/IGMLcreation/LoreMind) on GitHub.
- The source code of the installers is fully public and auditable.
- You can add a one-time exception for the download folder.
