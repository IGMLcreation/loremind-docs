# Utilisation des dossiers et sous-dossiers

Les **dossiers** sont les noeuds de l'arborescence de votre Lore. Ils servent
uniquement à organiser : ce sont les pages qui contiennent réellement le
contenu de votre univers.

## Lancer la création

Depuis l'écran d'accueil de votre Lore, dans la section **Dossiers** :

- Cliquez sur le bouton **Nouveau dossier** à droite du titre de la section.
- Si vous n'avez encore aucun dossier, le bouton **Créer votre premier dossier**
  au centre de l'écran fait la même chose.

Vous pouvez aussi créer un sous-dossier depuis la vue d'un dossier existant —
le parent sera alors pré-sélectionné dans le formulaire.

![Premier dossier](/img/screenshots/lore/dossiers/Premier_dossier.png)

## Le formulaire

Une page **Créer un nouveau dossier** s'ouvre avec les champs suivants :

- **Nom du dossier** *(obligatoire)* — par exemple `Personnages`, `Cités`,
  `Factions`. Préférez un nom de **catégorie** plutôt que d'entité concrète :
  un dossier `Cités` contiendra plusieurs pages cités, pas l'inverse.
- **Dossier parent** *(optionnel)* — laissez vide pour créer un dossier à la
  racine du Lore, ou choisissez un dossier existant pour créer un sous-dossier.
- **Icône** — choisissez parmi la grille d'icônes proposées. C'est un repère
  visuel dans la sidebar et la grille des dossiers, sans impact fonctionnel.
- **Description** *(optionnel)* — quelques mots pour rappeler à quoi sert le
  dossier. Utile si vous revenez sur le projet après plusieurs semaines.

![Creation dossier](/img/screenshots/lore/dossiers/creation_dossier.png)

Validez avec **Créer le dossier**. Vous arrivez ensuite sur la vue du dossier,
prêt à créer de nouveaux dossiers, templates ou nouvelles pages.


![Vue dossier](/img/screenshots/lore/dossiers/Premier_dossier2.png)

## Création d'un sous-dossier

Plusieurs chemin sont possibles pour créer un sous-dossier :

1. Cliquer sur le bouton **+Dossier** violet en haut dans la liste.
Dans la page qui s'ouvre, le champ "dossier parent" est par défaut à racine mais vous pouvez
sélectionner un autre dossier existant qui deviendra le parent du nouveau dossier.

![bouton +dossier](/img/screenshots/lore/dossiers/bouton_dossier.png)

2. Directement depuis la liste : si vous survolez un dossier, vous aurez 2 petits boutons qui permettent
de soit créer un dossier, soit une page. Celui de gauche vous permez de créer le sous-dossier :

![bouton +dossier 2](/img/screenshots/lore/dossiers/bouton_dossier2.png)

:::tip
Contrairement au bouton +Dossier, le bouton de gauche vous permet de créer un dossier avec le bon 
dossier parent associé.
:::

3. Et enfin, vous pouvez vous rendre dans votre dossier créer, vous aurez alors une page ressemblant à ça :

![Vue dossier](/img/screenshots/lore/dossiers/vue_dossier.png)

Cliquer sur le bouton **+ Nouveau sous-dossier** vous permettra de créer un sous-dossier avec le 
dossier parent associé, comme pour la deuxième méthode.

## Modification / suppression d'un dossier

Et enfin, vous pouvez modifier un dossier en cliquant sur le bouton "Modifier" en haut à droite, ou simplement
le supprimer en cliquant sur le bouton "Supprimer". Pour cela, il faut être sur le dossier.

## Bonnes pratiques

- **Dossiers = catégories, pages = entités.** Évitez un dossier qui contiendrait
  une seule page à long terme : si l'entité est seule, mettez-la directement
  à la racine.
- **Profondeur 3 maximum** dans 90% des cas. Au-delà, la navigation devient
  pénible — voir [Hiérarchie et navigation](/docs/lore/hierarchie).
- **Renommer plus tard est sans risque.** Les liens entre pages sont stockés
  par identifiant, pas par nom : vous pouvez réorganiser librement.

:::tip
Si vous hésitez sur la structure, commencez plat (tous les dossiers à la
racine) et regroupez en sous-dossiers seulement quand un dossier dépasse une
dizaine de pages.
:::
