# Création d'un nouveau lore

Lorsque vous lancez l'application pour la première fois, vous n'avez pas de lore existant.
Cliquez sur le bouton "Nouveau Lore" afin de créer votre premier univers :

![Bouton nouveau lore](/img/screenshots/lore/creation-lore/bouton_nouveau.png)

Une fenêtre va apparaitre :

![modale création lore](/img/screenshots/lore/creation-lore/modale_lore.png)

- **Nom de l'univers** — le nom de votre univers (ex: "Vermoria", "Eberron homebrew").
- **Description** — une brève description de votre univers. Décrivez l'histoire de ce dernier ainsi que son
ambiance par exemple.

Une fois votre lore créé, vous pouvez arrivez sur la page d'accueil de ce dernier avec les dossiers
et les pages qu'il contient :

![accueil lore](/img/screenshots/lore/creation-lore/accueil_lore.png)

Vous pouvez alors créer votre premier ou votre premier template.

:::warning
Avant de créer une page, il faut créer soit le dossier parent, soit le template.
Ceci dit, vous pouvez quand même lancer la création de page. Le programme vous proposera de créer
le dossier et le template.
:::

Je vous propose de vous rendre à la partie [création de dossier](/docs/lore/creation-dossier)
