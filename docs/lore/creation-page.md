# Création d'une page

Une **page** est une fiche typée par un [template](/docs/lore/templates) :
c'est elle qui porte le contenu réel de votre univers (une cité, un PNJ
historique, une faction…).

## Prérequis

Avant de créer une page, il vous faut :

1. **Au moins un template** dans le Lore — sinon il n'y a aucune structure
   de fiche à appliquer. Voir [Templates et types de champs](/docs/lore/templates).
2. **Au moins un dossier** — toute page doit être rangée quelque part. Voir
   [Création d'un dossier](/docs/lore/creation-dossier).

Si l'un des deux manque, le formulaire vous proposera un raccourci pour le
créer avant de revenir.

## Lancer la création

Depuis la vue d'un dossier, cliquez sur **Nouvelle page**. Vous arrivez sur
l'écran **Créer une nouvelle Page**.

## Le formulaire

- **Titre de la page** *(obligatoire)* — le nom de l'entité : `Maître Eldrin`,
  `La Cité d'Argent`… C'est ce qui s'affichera dans la sidebar et les liens
  entre pages.
- **Template** *(obligatoire)* — la structure de la fiche. Chaque template est
  présenté sous forme de carte avec son nom et sa description ; cliquez pour
  sélectionner. Le choix du template **détermine les champs** de la page (texte,
  image, etc.).
- **Dossier de destination** *(obligatoire)* — où ranger la page dans
  l'arborescence. Pré-sélectionné si vous arrivez depuis un dossier.

## Deux modes de création

Une fois titre, template et dossier renseignés, deux boutons sont disponibles :

### Créer la page (manuel)

Crée une page **vide** avec les champs du template prêts à être remplis. Vous
arrivez directement en mode édition.

À privilégier quand vous avez déjà l'entité en tête et que vous voulez écrire
le contenu vous-même.

### Créer avec l'IA (assistant)

Ouvre un **drawer de chat** avec un assistant IA en mode wizard. Vous décrivez
l'entité en langage naturel (`une cité naine creusée dans une montagne, dirigée
par un conseil d'aînés`), l'IA pose éventuellement des questions, puis
**pré-remplit les champs du template** à partir de la conversation.

À privilégier pour :

- Démarrer rapidement quand vous n'avez qu'une idée vague.
- Obtenir un brouillon cohérent que vous viendrez ensuite éditer.
- Maintenir une cohérence avec le reste du Lore (l'IA s'appuie sur le contexte
  de l'univers).

:::tip
Vous pouvez tout à fait créer la page vide puis utiliser la **génération IA
champ par champ** ensuite — c'est utile quand vous voulez contrôler chaque
section indépendamment plutôt que tout générer d'un coup.
:::

## Et après ?

Une fois la page créée, vous pouvez :

- **Remplir / éditer les champs** — chaque champ s'édite indépendamment.
- **Lier d'autres pages** via le picker de liens (les liens sont stockés par
  identifiant, donc résistants aux renommages).
- **Déplacer la page** dans un autre dossier si vous réorganisez votre Lore.
