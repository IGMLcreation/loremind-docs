# Génération de pages Lore

Au-delà du chat libre, LoreMind permet de **générer ou regénérer un champ
spécifique** d'une page Lore via un bouton dédié.

## Workflow

1. Ouvrir une page Lore (ex: "Cité de Vaeril")
2. À côté de chaque champ TEXT, un bouton **✨ Générer**
3. L'IA reçoit en contexte : la page elle-même + les autres champs déjà remplis + le Lore parent
4. Elle propose un texte pour le champ ciblé
5. Vous pouvez **accepter**, **régénérer** (nouvelle proposition), ou **affiner**

## Pourquoi par champ et pas en bloc ?

Génération ciblée :

- L'IA reste cohérente avec ce qui est déjà là
- Vous gardez le contrôle field par field
- Vous pouvez démarrer avec une page presque vide et la remplir progressivement

## Champs concernés

- **TEXT** uniquement — les champs IMAGE / NUMBER / KEY_VALUE_LIST ne sont pas générés par l'IA
- L'IA peut s'appuyer sur la valeur des autres champs (texte ou numérique) comme contexte

## Astuces

- **Remplissez quelques champs à la main d'abord** — donne du grain à moudre à l'IA
- **Régénérez plusieurs fois** — la première proposition n'est pas toujours la meilleure
- **Affinez par chat** — si une génération est presque bien, ouvrez le chat sur la page et demandez l'ajustement précis
