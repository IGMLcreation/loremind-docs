# Bienvenue dans la documentation de DM Loremind

**DM Loremind** est un assistant de jeu de rôle auto-hébergé qui combine gestion de campagnes, fiches de personnages, lore
et IA (locale ou cloud) pour vous aider à préparer vos parties.

## Caractéristiques principales

<IconFeatureList>
  <IconFeature Icon={Dices} title="Multi-systèmes">gérez plusieurs campagnes avec leurs propres règles JDR.</IconFeature>
  <IconFeature Icon={Sparkles} title="IA locale ou cloud">Ollama en local pour une confidentialité totale, ou une IA cloud au choix : 1min.ai, Mistral, Google AI Studio, OpenRouter.</IconFeature>
  <IconFeature Icon={ShieldCheck} title="100 % auto-hébergé">vos données restent sur votre machine.</IconFeature>
  <IconFeature Icon={Download} title="Installation double-clic ou Docker">installeur Windows (.msi) ou AppImage Linux en double-clic, ou Docker pour un homelab.</IconFeature>
  <IconFeature Icon={RefreshCw} title="Mises à jour">notification intégrée en mode bureau (releases GitHub), ou automatiques via Watchtower en Docker.</IconFeature>
</IconFeatureList>

## Par où commencer ?

- **Le plus simple — installeur double-clic :** → [**Télécharger DM Loremind**](/download) (Windows `.msi` / Linux `.AppImage`, rien d'autre à installer).
- **Auto-hébergement / homelab (Docker) :**
  - → [Guide d'installation Docker — Windows](/docs/installation/docker#windows)
  - → [Guide d'installation Docker — Linux](/docs/installation/docker#linux)
- **Vous voulez voir avant d'installer ?** → [Démo en ligne](https://loremind-demo.igmlcreation.fr)
- **Vous cherchez le code source ?** → [GitHub](https://github.com/IGMLcreation/LoreMind)
