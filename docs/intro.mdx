# Bienvenue sur LoreMind

**LoreMind** est un assistant de jeu de rôle auto-hébergé qui combine gestion de campagnes, fiches de personnages, lore et IA locale pour vous aider à animer vos parties.

## Caractéristiques principales

- 🎲 **Multi-systèmes** — gérez plusieurs campagnes avec leurs propres règles JDR.
- 🧠 **IA locale ou distante** — Ollama embarqué ou modèles OpenAI/Anthropic au choix.
- 🔒 **100 % auto-hébergé** — vos données restent sur votre machine.
- 🐳 **Installation Docker en une commande** — Windows, Linux, macOS.
- 🔄 **Mises à jour automatiques** optionnelles via Watchtower.

## Architecture

LoreMind est composé de trois services qui communiquent via Docker Compose :

- **Core** — backend Java/Spring Boot (API métier, persistance Postgres + MinIO).
- **Brain** — service Python/FastAPI dédié à l'IA (orchestration LLM, RAG).
- **Web** — interface utilisateur Angular.

Optionnellement, un quatrième service **Ollama** peut être lancé pour servir les modèles LLM en local.
