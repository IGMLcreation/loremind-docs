# Catalogue d'objets

Un **catalogue d'objets** est une collection d'objets propre à votre campagne :
boutique d'un marchand, table de butin, contenu d'un coffre… Vous créez autant
de catalogues que nécessaire.

## Le catalogue

- **Nom** (obligatoire)
- **Description**
- **Icône**

Chaque catalogue contient ensuite sa propre liste d'objets.

## Un objet

- **Nom** (obligatoire)
- **Prix** — texte libre, par exemple `50 po` ou `1 200 pièces d'or`
- **Catégorie** — sert au regroupement, par exemple `Armes`, `Potions`,
  `Parchemins`
- **Description** — rédigée en markdown

:::info[Pas de champ rareté]
Il n'existe pas de champ « rareté » dédié. Si vous voulez distinguer les objets
rares, utilisez la catégorie ou mentionnez-le dans la description.
:::

## Génération IA

Un champ de **prompt** accompagné d'un bouton **« Générer »** permet à l'IA de
pré-remplir le catalogue : nom, description et la **liste d'objets**. Vous pouvez
ensuite ajuster, compléter ou supprimer les entrées proposées.

## Affichage

En vue, les objets sont **groupés par catégorie**, ce qui transforme le
catalogue en inventaire de boutique lisible.

## Supprimer un catalogue

La suppression d'un catalogue se fait directement depuis la liste des
catalogues de la campagne.
